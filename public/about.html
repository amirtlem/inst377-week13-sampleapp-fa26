About This Project

Art Explorer is a web application that allows users to search artwork from the Metropolitan Museum of Art API, visualize artwork categories, and save favorite pieces into a personal collection using a Supabase database. In order to find your artworks, insert a word like "cat" and corresponding artworks will pop up.

This project was built using:

  • Node.js
  • Express.js
  • Supabase
  • Chart.js
  • Swiper.js
